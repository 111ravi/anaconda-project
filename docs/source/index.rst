================
Anaconda Project
================

Anaconda Project encapsulates data science projects and makes them easily portable. Anaconda Project automates setup steps such as installing the right
packages, downloading files, setting environment variables, and running commands.

Anaconda Project makes it easy to reproduce your work, share projects with others, and run them on different platforms, and even simplifies deploying them on servers.

Anaconda Project is supported and offered by Continuum Analytics and contributors under a 3-clause BSD license.
